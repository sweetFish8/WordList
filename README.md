# WordList

自分で英単語と日本語訳を登録し、クイズ形式で復習できる SwiftUI 製の単語帳アプリです。

## Features

- 英単語と日本語訳の登録
- SwiftData による端末内保存
- 登録済み単語の一覧表示
- 英単語から日本語訳を確認するクイズ
- 未入力時のバリデーション

## Tech Stack

- Swift
- SwiftUI
- SwiftData
- NavigationStack

## Screens

- ホーム: クイズまたは単語一覧を選択
- 単語一覧: 登録内容の確認と新規追加
- クイズ: 答えの表示・非表示と次問題への移動

## Getting Started

1. このリポジトリを clone します。
2. `WordList.xcodeproj` を Xcode で開きます。
3. `WordList` scheme を iOS Simulator または実機で実行します。
